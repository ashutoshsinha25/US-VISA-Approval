# US Visa Approval 


dataset used for the model : [link](https://www.kaggle.com/datasets/moro23/easyvisa-dataset)

## How to setup the code?

```bash
conda create -n visa python=3.8 -y
```

```bash
conda activate visa
```

```bash
pip install -r requirements.txt
```
